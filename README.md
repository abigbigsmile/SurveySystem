## 简易调查问卷系统

### 功能特点

> 面向用户的问卷搜索查看、发布问卷功能；面向问卷管理员的用户管理、问卷管理功能

### 环境要求

> Eclipse + Tomcat 

### 使用技术

> Servlet + Jsp + Bootstrap + jQuery

### 安装部署

> github上clone下来，import进Eclipse，部署启动
